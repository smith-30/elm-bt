# elm-bt

elm battle timer
application page is [here](https://smith-30.github.io/elm-bt/)

## Elm Develop Environment

fork from https://github.com/ababup1192/elm-dev-env

## Install

```shell
$ npm i
```

## Launch develop server

```shell
$ npm start
```

## Test

```shell
$ npm test
```

## Build

```shell
$ npm run build
$ open dist/index.html
```
